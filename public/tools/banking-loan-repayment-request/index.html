Requesting Repayment Gracefully

Talking about money with friends or family can be awkward. Using a structured request helps remove the emotion from the conversation while staying clear about the facts. Whether you need the money urgently or just want to close out an old agreement, a professional note helps set the right expectations.

FAQ

Which tone should I choose?

Use **Gentle** for a first reminder or a close friend. Use **Neutral** for a more formal or standard follow-up. Use **Firm** if you've already asked multiple times or if the deadline is long overdue.

What if they can't pay the full amount?

You might want to suggest a payment plan if the total amount is large. This shows flexibility while still ensuring you get your money back eventually.
