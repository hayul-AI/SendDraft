When to use this

  • Request money back for a product that didn't meet expectations.
  • Ask for a refund due to a service failure or non-delivery.
  • Correct a billing error such as a duplicate charge.

Tips

  • Always include your order number or reference if you have one.
  • Be clear about whether you want a full refund or store credit.
  • If you've already sent the item back, mention the return status.

Note: SendDraft provides templates for convenience. This tool does not provide legal or professional advice.

Professional Refund Requests

A well-written refund request increases the likelihood of a quick resolution. By providing all necessary details—including the amount, date, and reason—you make it easier for the support team to process your claim.

FAQ

Which tone should I use?

Use **Gentle** if it's your first time reaching out or if you have a good history with the company. Use **Neutral** for standard professional requests. Use **Firm** if your previous attempts were ignored or if the issue is a clear error on their part.

Do I need to return the item first?

It depends on the company's policy. Most will require the item to be returned or for a return to be initiated before a refund is issued. Use the 'Return status' field to let them know where things stand.
